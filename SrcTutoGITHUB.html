Page HTML Source Tuto GITHUB

Ceci est une page HTML Src pour le tuto comment se servir de GITHUB

(Avant de commencer dans le texte qui suit, enregistrez vous sur github et git. comment faire? : git config --global user.name"(username)" ensuite git config --global user.email "(adresse email)"

Pour commencer dans GIT creer un dossier PARENT (decidez du nom)
ensuite (dans la cmd) tapez: cd Bureau/(nom du dossier PARENT) puis entrez

Ensuite tapez cette commandes: git init ensuite creer vos fichiers HTML dans votre dossier PARENT puis enchainé sur ces commandes :

git add . (. = tous les fichiers present dans le dossier) git commit -m "(votre msg d'ajout ou autre) sinon faite git commit (entrez) ecrivez votre msg puis (controle X) O (entrez) puis

git fetch https://www.github.com/(username)/(nom de votre projet github) (entrez) si ca ne marche pas cherchez sur youtube ou google (google est ton ami;D) puis enchainé sur cette commande : git push

Mettez votre pseudo du site github puis votre mot de passe (github) puis actualisez la page et regardez le resultat :D voila le tuto est fini, je vous dit au prochain tuto :D
